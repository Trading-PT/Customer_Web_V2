# shared/ui 는 가장 작은 단위인 공통 컴포넌트를 모아둔 곳입니다.
## 공통 컴포넌트가 추가되는 경우 여기에 설명을 추가해주세요.
## 공통 컴포넌트의 속성이 변경되는 경우 반드시 팀원들에게 공유해주세요.

---------------------------------------------------------------------------

## CustomBox
- variant 종류: normal, notice, alert, highlight 
- props 조절 가능 (width, height, bg-color, p 등...)

## CustomButton
- variant 종류: normal-full, normal-clean, error, disable, success
- props 조절 가능 (width, height, bg-color, p 등...)

## CustomCheckbox
- variant는 없음
- props 조절 가능 (width, height, color, icon 등...)

## CustomDivider
- variant 종류: horizen, vertical
- props 조절 가능 (width, height, bg-color, m 등...)

## CustomDropdownButton
- variant는 없음
- props 조절 가능 (width, height, bg-color, p 등...)

## CustomInputField
- variant 종류: normal, length-limit-bottom-right, with-button-right
- props 조절 가능 (최대글자수, 오류메시지, 입력값 제어/검증 등...)

## CustomModal
- variant 종류: normal, with-x-button, with-ok-button, with-ok-no-button
- props 조절 가능 (width, height, bg-color, p 등...)

## CustomTable
- variant는 없음
- props 조절 가능 (width, height, bg-color, p 등...)

## CustomTextArea
- variant 종류: normal, length-limit-bottom-right
- props 조절 가능 (최대글자수, 최대 row 수, 오류메시지, 입력값 제어/검증 등...)
